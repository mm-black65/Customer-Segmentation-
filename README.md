# 🛍️ Customer Segmentation using K-Means Clustering

## 📌 Overview

This project focuses on segmenting customers based on their purchasing behavior using the K-Means clustering algorithm. By analyzing features such as annual income and spending score, customers are grouped into different segments to help businesses understand their target audience.

---

## 🎯 Objective

* Identify distinct customer groups based on behavior
* Understand spending patterns
* Provide insights for better marketing strategies

---

## 📊 Dataset

* Mall Customer Dataset
* Features used:

  * Annual Income (k$)
  * Spending Score (1-100)

---

## 🧠 Methodology

1. Data loading and exploration
2. Feature selection
3. Applying K-Means clustering
4. Using the Elbow Method to determine optimal clusters
5. Assigning cluster labels to customers
6. Interpreting clusters into meaningful customer segments

---

## 🔍 Key Features

* Unsupervised learning using K-Means
* Data visualization using Matplotlib and Seaborn
* Cluster analysis and interpretation
* Conversion of clusters into meaningful business insights

---

## 📈 Results

Customers were grouped into distinct segments such as:

* High Income – High Spending (VIP Customers)
* High Income – Low Spending (Potential Targets)
* Low Income – High Spending
* Low Income – Low Spending

These insights can help businesses design targeted marketing strategies.
